# Analýza dat databáze StackOverflow
Projekt slouží k procvičení SQL dovedností na reálných datech.
Cílem je praktické vytváření SQL dotazů v rámci interaktivního simulátoru Yandex.Practicum, kde bylo možné kód přímo vykonávat a okamžitě kontrolovat výsledky. Projekt obsahuje 20 úloh zaměřených na SQL dotazy nad databází StackOverflow (rok 2008).

## Data
- Databáze obsahuje 6 tabulek
- Zahrnuje informace o uživatelích, jejich publikacích (včetně typů), hlasech (včetně typů) a odznacích přidělených za publikace 
- ER-diagram databáze je součástí projektu (viz obrázek níže):

## Technologie a nástroje
- PostgreSQL  
- SQL (včetně CTE, okenních funkcí a JOIN)
- Interaktivní simulátor Yandex.Practicum (umožnil testování a okamžité vyhodnocování výsledků dotazů)

## Výsledky
- Výsledky zahrnují výstupy jednotlivých SQL dotazů, včetně agregací, filtrování a řazení dat.
