# Tvorba dashboardu o TED-konferencích – Tableau vizualizace
Cíl projektu: Na základě dat ze stránek konferencí TED vytvořit dashboard s různou analytikou o historii vystoupení, tématech, autorech a kategoriích.

## Data
- Pět CSV souborů o TED-konferencích
- Obsahují: data o vystoupení (id, URL, název, popis, datum, délka, počet zobrazení, jazyk, id konference atd.), seznam vystoupení (id, název a země konference), seznam autorů (jméno, obor autora)

## Technologie
- Tableau (Actions, Sets, Parametry, Top-N, Filtry, různé typy grafů, shrnutí pomocí prezentace)

## Postup a výsledky – Prezentace "Analytika konferencí TED", která obsahuje:
- Dashboard "Historie vystoupení": vystoupení dle zemí, dle let; procento vystoupení dle kategorií za každý rok; filtr Datum vystoupení; Action podle země vystoupení
- Dashboard "Témata vystoupení": podíl vystoupení podle témat; doba potlesku; doba smíchu; spojení délky a počtu zobrazení
- Dashboard "Autoři vystoupení": sféra práce; autoři podle počtu vystoupení; vystoupení podle počtu zobrazení; rozložení počtu autorů podle počtu vystoupení
- Dashboard "Analytika kategorií konferencí": kategorie, počet vystoupení podle roku, analytika kategorie "Jiné"
