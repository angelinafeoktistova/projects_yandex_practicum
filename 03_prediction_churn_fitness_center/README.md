# Predikce odlivu klientů sítě fitness center – klasterizace (K-Means)

Řetězec fitness center zaznamenává v poslední době zvýšený odliv klientů. Cílem projektu bylo identifikovat rizikové skupiny klientů s vysokou pravděpodobností odchodu a odhalit faktory, které k tomu vedou.

## Data
- Zdroj: CSV soubor s klientskými anketami
- Obsah: demografické údaje, informace o docházkách, status odchodu klienta apod.

## Technologie
- Python (Pandas, Matplotlib, Seaborn, Plotly, NumPy, Scikit-learn, SciPy)
- Jupyter Notebook

## Postup analýzy
- Kontrola kvality dat
- EDA: rozložení proměnných, korelace a rozdíly mezi odcházejícími a zůstávajícími klienty
- Základní predikční modely: Logistic Regression, Random Forest Classifier
- K-Means

## Výsledky
- Vytvoření profilu typického klienta
- Klasterizace klientů a identifikace nejrizikovějších skupin
- Identifikace klíčových faktorů pro snížení odlivu
- Příprava doporučení pro udržení klientů

## Závěr
- Projekt ukazuje, které skupiny klientů jsou nejvíce ohrožené odchodem, a poskytuje doporučení pro jejich udržení
