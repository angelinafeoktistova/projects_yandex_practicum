# Projekty z rekvalifikačního kurzu "Data Analyst" (Yandex Practicum)
Níže jsou uvedeny klíčové projekty, které jsem zpracovala v rámci rekvalifikačního kurzu Data Analyst.

**Yandex Practicum** je mezinárodní vzdělávací platforma známá v Rusku, Evropě a USA. Je jedním z vzdělávacích projektů společnosti Yandex.


## Základní dovednosti a nástroje, které jsem během rekvalifikace získala:
- Python (Pandas, NumPy, Matplotlib, Seaborn, Plotly, SciPy, Scikit-learn)
- SQL (včetně CTE, okenních funkcí, JOIN), PostgreSQL
- Kontrola kvality dat, ETL, EDA
- Popisná a aplikovaná statistika, základy testování hypotéz, A/B testování
- Základy strojového učení (ML) 
- Základní vizualizace dat
- Základy tvorby dashboardů v Tableau
