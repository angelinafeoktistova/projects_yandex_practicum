# Analýza odlivu zákazníků v bance. Segmentace klientů
Regionální banka zaznamenala výrazný nárůst odchodu klientů. Cílem projektu bylo identifikovat segmenty klientů s nejvyšším odlivem, popsat jejich charakteristiky a navrhnout doporučení pro jejich udržení.

## Data
- Zdroj: soubor CSV s informacemi o klientech několika poboček banky
- Obsah: demografické údaje, informace o účtech, status odchodu klientů

## Použité technologie
- Python (Pandas, NumPy, Matplotlib, Seaborn, SciPy, phik)
- Jupyter Notebook

## Postup analýzy
- Kontrola kvality dat  
- Analýza celkového rozložení dat, porovnání odcházejících a zůstávajících klientů a korelační analýza
- Testování statistických hypotéz  
- Segmentace odcházejících klientů pro cílenou analýzu

## Výsledky
- Profil typického a odcházejícího klienta
- Identifikace klíčových segmentů odcházejících klientů
- Doporučení pro udržení klientů
- Ověření statistických hypotéz souvisejících s rozdíly mezi odcházejícími a zůstávajícími klienty

## Závěr
- Analýza identifikovala hlavní faktory ovlivňující odchod klientů a umožnila navrhnout cílené kroky k udržení klíčových segmentů.
