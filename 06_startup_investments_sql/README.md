# Analýza dat databáze Startup Investments
Projekt slouží k procvičení SQL dovedností na reálných datech.
Cílem je praktické sestavování SQL dotazů v interaktivním tréninkovém prostředí Yandex.Practicum, kde bylo možné kód přímo vykonávat a ověřovat jeho výsledky. Obsahuje 23 úkolů zaměřených na sestavení dotazů do databáze Startup Investments z Kaggle (https://www.kaggle.com/justinas/startup-investments). 

## Data
- Databáze obsahuje 7 tabulek
- Obsahuje informace o startupových firmách, venture fondech, akvizicích mezi firmami, investičních kolech, investicích venture fondů do startupů, zaměstnancích startupů a o vzdělání zaměstnanců firem

## Technologie a nástroje
- PostgreSQL  
- SQL SQL (včetně CTE, agregačních funkcí, JOIN, CASE a GROUP BY)
- Interaktivní simulátor Yandex.Practicum umožnil testování a okamžité vyhodnocování výsledků dotazů.  

## Výsledky
- Výsledky zahrnují agregace, řazení, filtrování a další operace pro jednotlivé úkoly.
