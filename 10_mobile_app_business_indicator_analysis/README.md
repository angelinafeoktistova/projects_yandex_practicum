# Analýza obchodních ukazatelů mobilní aplikace
Mobilní aplikace investovala do reklamy, ale projekt vykazuje ztráty. Cílem analýzy je zjistit, proč je návratnost investic nízká, a navrhnout způsoby, jak ji zlepšit. Data o uživatelích, objednávkách a reklamních výdajích byla vyčištěna a připravena pro analýzu. Zkoumaly se klíčové metriky marketingu a produktu, jako CAC, LTV, ROI, CR a RR, a výsledky byly vizualizovány pro hodnocení výkonu jednotlivých kanálů a přípravu doporučení.

## Data
- Zdroj: 3 CSV soubory
- Obsah: logy o aktivitě uživatelů, objednávky, data o reklamních nákladech

## Technologie
- Python (Pandas, NumPy, Seaborn)
- Výpočty unit-ekonomiky: CAC, LTV, ROI, CR, RR
- Jupyter Notebook

## Postup analýzy
- Kontrola kvality dat
- Příprava uživatelských profilů,
- Zkoumaní rozložení dat podle zemí, zařízení a zdrojů návštěv
- Výpočet rozložení a dinamiky reklamných nákladů, CAC, LTV, ROI, CR, RR, doby návratnosti (s využitím předem připravených funkcí od lektorů kurzu)
- Vizualizace metrik
- Vyhodnocení výkonu reklamních kanálů

## Výsledky
- Segmentace uživatelů podle zemí, zařízení a zdrojů příchodu.
- Výsledky jednotlivých reklamních kanálů
- Spočítány CAC, ROI, doba návratnosti, konverze a LTV.
- Identifikace slabých míst strategie. Doporučení ke zlepšení návratnosti investic
